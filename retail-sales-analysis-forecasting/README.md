# Retail Sales Analysis & Forecasting

A portfolio project that analyzes sales performance, profitability, and monthly forecasting using **Python, SQL, and Power BI/Tableau-ready outputs**.

## Business problem
Retail businesses need a clear view of revenue trends, product performance, and future sales expectations to support planning and decision-making.

## Dataset
This repository includes a **synthetic retail sales dataset** (`data/raw/retail_sales.csv`) that is safe for GitHub.

## Tools used
- Python
- SQL
- Power BI or Tableau
- Pandas / NumPy / Matplotlib / scikit-learn

## Project workflow
1. Load and validate transaction data
2. Analyze sales, profit, region, and product patterns
3. Build monthly summaries
4. Create a simple forecasting baseline and linear model
5. Export dashboard-ready tables and visuals

## Key questions answered
- Which categories and regions drive the most revenue?
- How do discounts affect profitability?
- Which products perform best?
- How well does a simple forecast perform on monthly sales?

## Repository structure
- `data/raw/` raw CSV data
- `data/cleaned/` cleaned or aggregated outputs
- `notebooks/` exploratory and presentation-ready notebook
- `sql/` schema and analysis queries
- `src/` reusable Python analysis script
- `dashboard/` dashboard layout notes
- `images/` chart screenshots
- `reports/` final summary files
