# Insurance Claims & Fraud Analytics

A portfolio project that explores claim behavior, fraud indicators, and operational patterns using **Python, SQL, and Tableau-ready outputs**.

## Business problem
Insurance providers need to detect suspicious claim behavior early so they can reduce fraud losses and improve investigation efficiency.

## Dataset
This repository includes a **synthetic insurance claims dataset** (`data/raw/insurance_claims.csv`) that is safe to share publicly.

## Tools used
- Python
- SQL
- Tableau
- Pandas / NumPy / Matplotlib

## Project workflow
1. Load and validate claim records
2. Explore claim severity, incident type, and settlement patterns
3. Measure fraud rate across segments
4. Export summary outputs for dashboarding
5. Convert insights into business recommendations

## Key questions answered
- Which incident types have the highest fraud rate?
- Do larger claims have a higher fraud likelihood?
- Which regions or tenure groups should be monitored more closely?
- What operational patterns affect claim handling?

## Repository structure
- `data/raw/` raw CSV data
- `data/cleaned/` cleaned or aggregated outputs
- `notebooks/` exploratory and presentation-ready notebook
- `sql/` schema and analysis queries
- `src/` reusable Python analysis script
- `dashboard/` dashboard layout notes
- `images/` chart screenshots
- `reports/` final summary files
